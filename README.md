# Wizz

## Overview

Wizz is a lightweight desktop expense tracker built with Electron.js. It provides a simple interface for recording income and expenses, organizing transactions into categories, and visualizing spending habits through charts. All data is stored locally using the browser's Local Storage API, making the application fast, lightweight, and easy to use without requiring an internet connection.

The project was created to strengthen my understanding of JavaScript fundamentals, DOM manipulation, Electron.js, and local data persistence while building a practical desktop application.

---

## Features

- Record income and expense transactions
- View all transactions in a dynamic table
- Edit existing transactions
- Delete transactions with confirmation dialogs
- Automatic expense breakdown using Chart.js
- Live summary cards displaying:
  - Current Balance
  - Total Income
  - Total Expenses
- Manage custom transaction categories
- Automatic local data persistence using Local Storage
- Cross-platform desktop application powered by Electron.js

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/wizz.git
```

Navigate into the project:

```bash
cd wizz
```

Install dependencies:

```bash
npm install
```

---

## Run

Start the Electron application:

```bash
npm start
```
